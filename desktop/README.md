# Focus-Fee Desktop

Electron + React desktop client for the Focus-Fee hackathon app.

## Setup

```bash
npm install
```

Create `desktop/.env`:
```env
VITE_BACKEND_URL=https://your-vercel-project.vercel.app
```

## Run (dev)

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Run packaged entry (after build)

```bash
npm run start
```
